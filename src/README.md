# 🎬 IMDb Movie Analysis

## 📌 Overview

This project analyzes an IMDb movie dataset using Python to identify trends in movie budgets, genres, and IMDb ratings. It performs data cleaning, exploratory data analysis (EDA), statistical analysis, and visualization.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Budget vs IMDb Rating Analysis
- Genre-wise Average IMDb Ratings
- Budget Distribution Visualization
- Correlation Heatmap
- Top 10 Highest Rated Movies
- Top 10 Highest Budget Movies
- Interactive Plotly Visualization

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

---

## 📊 Key Insight

The analysis found that movie budget has **very little correlation** with IMDb ratings, indicating that higher budgets do not necessarily result in higher-rated movies.
