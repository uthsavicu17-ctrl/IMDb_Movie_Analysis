# IMDb Movie Analysis Dashboard

An interactive dashboard built using Streamlit to analyze IMDb movie data.

## Features
- Search movies
- Filter by genre
- Filter by release year
- IMDb rating filter
- Interactive Plotly charts
- Budget analysis
- Genre analysis
- Correlation heatmap
- Download filtered dataset

## Technologies
- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib
- Seaborn
